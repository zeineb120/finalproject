# ✨ Sticky Tasks AI

Une application de liste de tâches moderne et minimaliste conçue pour ressembler à un tableau de notes autocollantes (Sticky Notes). 

![Version](https://img.shields.io/badge/version-1.0.0-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)

## 🚀 Fonctionnalités

- **Aspect Moderne & Minimal** : Design épuré avec des animations fluides.
- **Notes Autocollantes** : Les tâches s'affichent sous forme de Post-its avec des ombres douces et des rotations aléatoires.
- **Mode Sombre** : Basculement entre le mode clair et sombre avec mémorisation de votre préférence.
- **Filtres Intelligents** : Visualisez rapidement vos notes par statut (Toutes, Actives ou Terminées).
- **Stockage Local** : Vos tâches sont sauvegardées automatiquement dans votre navigateur.
- **Raccourcis Clavier** :
  - `Entrée` : Ajouter une nouvelle tâche.
  - `Supprimer` : Supprimer la dernière tâche de la liste.
